## Architecture Notes for Agents

- **Stack:** Next.js 16 + Convex backend + Anthropic Claude API
- **Classification pipeline:** V4 at `src/v4/`, API route at `src/app/api/v4-analyze/`
- **Shared reference library:** `src/lib/references/` — tag-based resolver for document classification
- **Filing flow:** `src/lib/bulkQueueProcessor.ts` → `/api/v4-analyze`
- **Chat system:** Claude Haiku 4.5 via `@anthropic-ai/sdk`, tools in `src/lib/tools/`
- **Build check:** Always run `npx next build` before committing
- **Commit rule:** Push to GitHub after each change for code review ease
