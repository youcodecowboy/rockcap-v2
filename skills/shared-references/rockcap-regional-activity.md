# RockCap regional activity

Regions where RockCap has done or is doing deals, for the **RockCap-active-in-region hook** (rung 4 of `hook-ladder.md`). Reliable and evidence-based; no personal claim required. Strong when the prospect's region matches a region RockCap has worked in.

**Provenance:** Alex Lundberg's record of where RockCap has been active, with verbatim corpus examples of the hooks she has sent. Canonical here. If deals close in new regions, update this file. **Hard rule: never name the client. Regions only. No counterparty names in cold outreach.**

## When to use

Trigger: the prospect's region (HubSpot `company_county` / `state` / `city`, or Companies House charge particulars showing where their schemes are) overlaps an entry below. The claim must be **current and true**; if RockCap's activity in the region is historic, reframe ("we have done a lot of work in [region] historically and always keen to do more") rather than implying current activity.

## Hook patterns

- **Pattern A (closed):** "we have done some business in [REGION]" / "we've done a fair bit of work in [REGION] but would love to do more".
- **Pattern B (in-progress):** "we are working on a couple of deals in [REGION] and would love to do more in the area".
- **Pattern C (recent, only if current):** "we have been doing more in [REGION] over the last 12 months".
- **Pattern D (town pair, highly localised):** "We have done a handful of deals in and around [TOWN1] & [TOWN2]".

If a region triggers both this hook and a personal-geography hook (`sender-geography.md`), default to leading with the personal-geography angle and optionally append the RockCap-active clause.
